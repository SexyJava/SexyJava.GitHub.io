测试博客

2020.4.4博客搭建

  1. 安装node，git（git安装操作及配置请移步百度）（安装步骤无脑下一步）

  2. 安装完成后查看node和npm版本，已检查是否安装成功（node -v npm -v）

  3. 安装hexo install -g hexo-cli（全局安装hexo）

  4. hexo -v查看版本（如果找不到hexo命令，则编辑系统变量中的path新增node目录下包含hexo.cmd文件的路径）

  5. 新建一个博客根文件夹xxx

  6. 在根目录中右键点击git bash here

  7. 初始化hexo（hexo init）

  8. 初始化成功启动hexo（hexo s）

  9. 访问localhost:4000访问博客主页

  10. 关闭hexo（ctrl+c）

  11. 新建第一个博客（hexo n “我的博客”）

  12. xxx\source\_posts（博客存放目录）

  13. hexo clean 清理缓存

  14. hexo g 生成博客

  15. hexo s 检查是否成功

  16. GitHub创建仓库名为：GitHub账号名.github.io的仓库

  17. 安装git插件 cnpm install –save hexo-deployer-git

  18. 打开博客根目录下的_config.yml文件
